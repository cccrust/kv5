## 發展目標

v0.7 主要新增 SCAN 系列命令，實現基於遊標的鍵迭代，用於生產環境中的鍵掃描。

## 現有基礎

- `Store::dbsize()` 已展示迭代模式：`self.data.iter().filter(|e| !e.value().is_expired())`

## 功能規劃

### 1. SCAN 命令

Redis SCAN 用於增量迭代鍵空間，不會阻塞伺服器。

#### 1.1 命令格式

```
SCAN cursor [MATCH pattern] [COUNT count]
```

- **cursor**: 客戶端傳遞的遊標（第一次從 0 開始，0 表示開始新迭代）
- **MATCH pattern**: 可選，glob 模式過濾鍵名（`*` = 任意字符，`?` = 單字符）
- **COUNT count**: 可選，每次迭代返回的鍵數量提示（預設 10）

#### 1.2 回傳格式

```
1) cursor        # 下一個遊標，如果為 0 表示迭代完成
2) keys         # 本次返回的鍵陣列
```

#### 1.3 實現位置

在 `src/cmd.rs` 的 `handle_command()` 新增 case，在 `src/store/store.rs` 新增 `scan()` method。

#### 1.4 實現方式

```rust
// src/store/store.rs
pub fn scan(&self, cursor: usize, pattern: Option<&str>, count: usize) -> (usize, Vec<String>) {
    let all_keys: Vec<String> = self.data
        .iter()
        .filter(|e| !e.value().is_expired())
        .map(|e| e.key().clone())
        .collect();

    let total = all_keys.len();
    let start = cursor % total.max(1);
    let mut result = Vec::new();
    let mut next_cursor = cursor;

    for i in 0..count {
        let idx = (start + i) % total.max(1);
        let key = &all_keys[idx];

        match pattern {
            Some(p) if !matches_pattern(key, p) => continue,
            _ => result.push(key.clone()),
        }
    }

    let consumed = (start + count).min(total);
    next_cursor = if consumed >= total { 0 } else { cursor + count };

    (next_cursor, result)
}
```

### 2. SSCAN 命令

```
SSCAN key cursor [MATCH pattern] [COUNT count]
```

迭代 Set 中的成員。

實現位置：`src/store/store.rs` 的 `sscan()` method

### 3. HSCAN 命令

```
HSCAN key cursor [MATCH pattern] [COUNT count]
```

迭代 Hash 中的欄位（field）。

### 4. ZSCAN 命令

```
ZSCAN key cursor [MATCH pattern] [COUNT count]
```

迭代 Sorted Set 中的成員和分數。

## API 變更

### Store 新增方法

| 方法 | 回傳 | 說明 |
|------|------|------|
| `scan(cursor, pattern, count)` | `(usize, Vec<String>)` | 鍵空間掃描 |
| `sscan(key, cursor, pattern, count)` | `(usize, Vec<String>)` | Set 迭代 |
| `hscan(key, cursor, pattern, count)` | `(usize, Vec<String>)` | Hash 迭代 |
| `zscan(key, cursor, pattern, count)` | `(usize, Vec<String>)` | ZSet 迭代 |

### 新增命令

| 命令 | 類別 | 說明 |
|------|------|------|
| SCAN | Keyspace | 增量迭代鍵 |
| SSCAN | Set | 迭代 Set 成員 |
| HSCAN | Hash | 迭代 Hash 欄位 |
| ZSCAN | Sorted Set | 迭代 ZSet 成員 |

## 風險評估

1. **SCAN 效能**: 在大鍵空間上可能需要多次調用
2. **指標一致性**: SCAN 不保證在迭代期間的資料一致性
3. **COUNT 提示**: COUNT 只是提示，實際返回數量可能不同
